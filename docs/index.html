What is Bolt?

Bolt is a framework for building fast, responsive websites using Jekyll, Sass, and Gulp.

Simplicity

Bolt is minimalistic in structure and design, making it extremely easy to pick up and build upon.

Performance

Built for performance, Bolt uses Gulp to optimize CSS and image files for production builds.

Maintainability

Bolt makes use of HTML templating with Jekyll and Atomic CSS with Sass to promote better maintainability.

Blog Ready

Bolt makes it insanely easy to add blog posts without writing any additional code.
